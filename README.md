# clean.vim
A light-based neovim/vim colorscheme with few colors.

## Getting Started

### Vim
Copy clean.vim to your colors directory (usually ~/.vim/colors)

## Neovim

### [lazy.nvim](https://lazy.folke.io/installation)
```lua
return { NateEbling/clean.vim }
```
